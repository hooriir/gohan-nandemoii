## なぜこのアプリを作ったのか（制作背景）
我が家では、「今日のご飯何がいい？」と聞くと「なんでもいい」と言われてしまいます。
いつまでもご飯が決まらない問題をいつか解決したいと思っていました。
メニューを決めてくれるアプリはAIを含めていろいろありますが、
我が家の好みに合わせた（食べ慣れている）メニューを出してくれるアプリがあったらいいなと思い作成しました。
